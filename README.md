# Smart ToDo API

Smart ToDo API is a RESTful backend application built using Node.js and Express.
It allows users to register, login, and manage their personal tasks securely
using JWT authentication and MongoDB.

---

## Features
- User Registration & Login
- JWT Authentication
- Create, Read, Update, Delete (CRUD) Tasks
- MongoDB Atlas integration
- Swagger API Documentation
- Tested using Postman

---

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT (JSON Web Token)
- Swagger
- Postman
